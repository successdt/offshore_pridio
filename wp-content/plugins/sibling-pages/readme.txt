Generates and registers a widget that generates a list of links to sibling pages (pages that have the same parent page) 

== Description ==

* This plugin depends on setting some pages to have the same "parent" page.
* If page A, B and C all have the same parent page, then the plugin will generate a list of links to page A, B and C whenever you are on any one of those pages, or whenever you are on the parent page. 
* The parent page will be listed first in the list of siblings, then the sibling pages in apha order by page title.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Drag the "Sibling Pages" widget onto a display area in the 'widgets' admin page
